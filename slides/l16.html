One time setup

  • Create an account at Heroku.com
  • Install the Heroku Toolbelt
  • Login to heroku 
  • $ heroku login

Per-project setup

  • Create a git repo for your project 
    $ git init
  • add production gems to Gemfile
  • group :production do
  gem 'rails_12factor'
  gem 'pg'
end
  • move sqlite3 to development group
  • group :development, :test do
  gem 'sqlite3'
end
  • bundle without production 
  • bundle install --without production
  • create the heroku instance 
  • heroku create name-of-app

Per-commit steps

git add .
git commit -m "description"
git push heroku master

Heroku utility functions

  • running migrations
  • heroku run rake db:migrate
  • debuging
  • heroku logs
  • seeing your website
  • heroku open
