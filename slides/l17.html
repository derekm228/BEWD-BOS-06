Testing

A unit test is a bit of code that runs a discrete function (or unit) of production code and asserts that it is working correctly

why write tests?

  • find bugs and regressions
  • reduce the amount of manual testing
  • make code more readable and flexible
  • increase confidence and focus

Testing tools

  • minitest

Ruby Koans


require 'minitest/autorun'
require './fizzbuzz'

class TestFizzBuzz < Minitest::Test
  def test_fizz_buzz_3_is_fizz
    actual_result = fizzbuzz(3)
    expected_result = "fizz"
    assert_equal(actual_result, expected_result)
  end
end
